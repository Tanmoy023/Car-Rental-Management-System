# Car Rental Management System

A comprehensive web-based application for managing car rentals. This system allows customers to browse available cars, make bookings, and manage their reservations. It includes features for user authentication, booking management, and mock payment processing.

## Features

-   **User Authentication**: Register, Login, and Logout functionality.
-   **Car Inventory**: Browse a fleet of vehicles with detailed information and images.
-   **Booking System**:
    -   Check availability for specific dates.
    -   Real-time price calculation.
    -   "Pending" to "Confirmed" status workflow via mock payment.
    -   Cancellation option for pending bookings.
-   **Payments & Receipts**:
    -   Mock credit card payment processing.
    -   Printable booking receipts.
-   **User Dashboard**: View booking history and status.
-   **Responsive Design**: Built with Bootstrap 5 for mobile and desktop compatibility.

## Tech Stack

-   **Backend**: Django (Python)
-   **Frontend**: HTML, CSS, JavaScript, Bootstrap 5
-   **Database**: SQLite (Default) / PostgreSQL (Configurable)
